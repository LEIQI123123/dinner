# 🍚 点餐小程序 (Dinner Mini-Program)

一个可爱萌系、暖色调风格的微信小程序点餐应用。支持菜单分类浏览、菜品详情与规格选择、购物车实时算价、地址管理与订单结算，并内置**菜单管理**功能（可改价格、上传图片、增删菜品，本地持久化，无需后端）。

> 菜品图片默认用 emoji + 渐变卡片呈现（零图片依赖，最贴合萌系风格）；也可在「菜单管理」里上传真实照片。

## ✨ 功能特性

- 🏠 **首页**：渐变 hero 头图、分类快捷入口、促销 banner、店长推荐横滑、人气热销
- 📋 **菜单**：左侧分类 tab + 右侧菜品列表联动高亮，支持关键字浏览
- 🍽️ **菜品详情**：图片/emoji 头图、简介、辣度与份量规格选择、实时单价预览、加入购物车
- 🛒 **购物车**：数量加减、删除、清空，实时小计 / 满减优惠 / 配送费 / 合计，「再买 ¥X 减 ¥Y」提示
- 📦 **订单结算**：选择配送地址、填写备注、提交生成订单
- 📍 **地址管理**：地址列表、新增/编辑、删除、设为默认、手机号校验
- ⚙️ **菜单管理**：改价格、上传图片、增删菜品（本地持久化，全链路即时生效）

## 🎨 视觉风格

- 暖色调为主（主色 `#FF7E5F`）
- 圆角卡片、胶囊按钮、萌系 emoji 图标、轻量过渡动画
- 移动端布局，适配底部安全区

## 🧱 技术栈

- 微信小程序原生（WXML / WXSS / JS），无第三方框架、无构建步骤
- 前端内置 mock 数据，`utils/menuStore.js` 负责本地持久化与菜单 CRUD
- 底部导航使用 **custom-tab-bar + emoji 图标**（规避原生 tabBar 强制 png 图标的限制）

## 🚀 运行方式

1. 打开 **微信开发者工具** → 导入项目 → 目录选择 `dinner/`
2. AppID 选择 **「测试号」**（或填入你自己的小程序 AppID）
3. 编译后可在模拟器预览；手机预览请扫码

> 无真实后端，所有数据运行在本地，刷新不丢失。

## ✏️ 自定义菜单

1. 进入 **菜单页（📋）** → 右上角 **⚙️ 管理**，或首页右上角小齿轮
2. 点击任意菜品 **编辑** / **删除**，或左上角 **＋ 新增菜品**
3. 编辑表单支持：名称、分类、价格、原价、简介、标签、emoji 图标、配色、辣度/份量开关、推荐开关、**上传图片**
4. 保存后即时同步到首页、菜单、详情、购物车

## 🔧 优惠规则

- 满减（可在 `utils/mock.js` 的 `promos` 调整）：
  - 满 ¥30 减 ¥5
  - 满 ¥50 减 ¥10
  - 满 ¥88 减 ¥20
- 满 ¥30 免配送费（否则 ¥5）
